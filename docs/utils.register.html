This is the list of all Utils for register Blocks/Item with exemple

  1. Importe Utils.
    • (Java Code) import fr.reproject.core.RECore;

  2. Use The Utils
    • RECore.Utils.*

      • * it's the function you want

  3. Exemples
    • Register your items

      • public static void registerItems()* {
      RECore.Utils.registerItem(Your_Item_1, Your_Creative_Tab, Your_Looger);
      RECore.Utils.registerItem(Your_Item_2, Your_Creative_Tab, Your_Looger);
      RECore.Utils.registerItem(Your_Item_3, Your_Creative_Tab, Your_Looger);
      RECore.Utils.registerItem(Your_Item_4, Your_Creative_Tab, Your_Looger);
      }
      * You need to load this function aftre init this items in the forge preInit
    • Register your renders

      • public static void registerRenders()* {
      RECore.Utils.registerRender(Your_Item_1, Your_ModID, Your_Looger);
      RECore.Utils.registerRender(Your_Item_2, Your_ModID, Your_Looger);
      RECore.Utils.registerRender(Your_Item_3, Your_ModID, Your_Looger);
      RECore.Utils.registerRender(Your_Item_4, Your_ModID, Your_Looger);
      }
      * You need to load this function aftre are register this items in the forge preInit
